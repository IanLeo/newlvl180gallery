Put your real images in this folder and list them in `manifest.json`.

Example `manifest.json`:

```json
[
  "images/my-photo-1.jpg",
  "images/my-photo-2.png",
  "images/my-photo-3.webp"
]
```

Notes:
- Paths must be web paths relative to site root (start with `images/`).
- If `manifest.json` is empty, the gallery uses built-in placeholder images.
